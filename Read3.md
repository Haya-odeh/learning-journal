# what is git ?

It's a program that you use when you want to *track your work*,*save new changes* 
*view files that multipeople work on*

# what does git do ?

when using git it  :
1. snapshots
2. Do Local operations 
3. Track changes
4. minimize the loss of data 

some commands you can use :
* cd 
* mkdir
